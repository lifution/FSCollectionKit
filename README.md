## Requirements

* iOS 11+
* Swift 5
* Xcode 14+

## Installation

#### [CocoaPods](http://cocoapods.org) (recommended)

```ruby
pod 'FSCollectionKit'
```

#### [Carthage](https://github.com/Carthage/Carthage)

````bash
github "lifution/FSCollectionKit"
````
